
A monthly session run every third Thursday of each month where Scala
enthusiasts gather to solve various coding problems in
Scala. Participants are encouraged to bring their own laptops with the
appropriate Scala development tools already installed. Problems can be
tackled individually or in pairs according to each participant's own
preferences.

Venue: QOS.ch, rue Enning 4, 1003 Lausanne, Tel: 021 312 32 26

Solutions to various problems are discussed on our mailing list:
  http://groups.google.com/group/scala-enthusiasts-lausanne
